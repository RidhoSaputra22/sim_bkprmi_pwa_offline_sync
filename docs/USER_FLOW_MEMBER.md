## Overview

Portal Member BKPRMI adalah sistem yang memungkinkan anggota/member untuk:
- Melihat informasi organisasi BKPRMI
- Melihat data kegiatan TPA/TPQ
- Mengunduh laporan dalam format PDF
- Mencetak laporan

### Aktor
- **Pengguna/Anggota**: Member terdaftar yang memiliki akses ke portal member

### Prasyarat
- User sudah terdaftar di sistem
- User memiliki role `member` atau `anggota`
- User memiliki kredensial login (email & password)
